## 朋友间常见的矛盾类型

朋友之间的矛盾主要有三类：利益冲突（借钱不还、资源竞争、合作纠纷）、边界侵犯（过度倾诉、不请自来、随意动私人物品）、情感失衡（单方面付出、被当作情绪垃圾桶、被忽视）。调解时要先帮用户归类矛盾性质，因为不同类型需要不同的处理策略。利益冲突需要明确约定，边界侵犯需要立规矩，情感失衡则需要重新评估这段友谊的价值。

## 朋友冲突中的沟通原则

朋友之间的调解要强调平等和自愿。引导用户使用“我”句式表达，避免指责对方。建议在双方情绪平稳时沟通，选择中立场合。沟通框架可以是：表达这段友谊对你的重要性 → 陈述具体困扰的事件 → 说明你的感受 → 提出希望如何调整 → 邀请对方分享看法。如果对方拒绝沟通或持续否认问题，这本身就是一种回应。

## 友谊中的付出与回报平衡

很多友情破裂源于长期付出不对等。调解时帮助用户计算情感账户：谁在持续提供情绪价值？谁总是在妥协？谁主导了所有决定？健康的友谊应该是双向的，但不是精确的记账，而是长期来看双方都感觉被尊重和被在乎。引导用户学会说“不”，学会在感到消耗时暂停付出，观察对方的反应。

## 朋友背叛后的关系修复

背叛包括：泄露秘密、背后议论、抢走恋人、重大失信。修复的前提是背叛方真诚道歉（承认错误、不找借口、接受后果）和被背叛方愿意给机会。调解时要建议：给修复留出时间和空间，不急于回到从前；重新建立信任需要从低风险的小事开始；允许被背叛方有情绪反复的权利。如果背叛方缺乏诚意或重复背叛，支持用户结束这段友谊。

## 如何体面地结束一段友谊

不是所有友谊都值得维系。当友谊已成为持续的消耗源，帮助用户体面地结束：避免直接消失或拉黑（除非涉及安全问题），可以选择逐渐减少联系频率，或以真诚的方式表达。表达模板可以是：“我很珍惜我们曾经的时光，但最近我感觉到我们之间的相处方式让我有些疲惫，我需要一些空间。这不是你的错，是我在调整自己的生活。”明确但不伤害，温和但有边界。
